200_JythonExtensionProvider.py¶

This component implements the openHAB extension provider interfaces and can be used to provide symbols to a script namespace.

class scripts.core.components.200_JythonExtensionProvider.JythonExtensionProvider¶

Bases: java.lang.Class.forName

getDefaultPresets()¶

These presets will always get injected into the ScriptEngine on instance creation.

getPresets()¶

Returns the provided Presets which are supported by this ScriptExtensionProvider. Presets define imports which will be injected into the ScriptEngine if called by “importPreset”. Note: default preset names must also be in this list.

getTypes()¶

Returns the supported types which can be received by the given ScriptExtensionProvider. “Types” are just names and can refer to any object, either instance or class.

get(scriptIdentifier, name)¶

This method should return an Object of the given type. Note: get can be called multiple times in the scripts use caching where appropriate. The scriptIdentifier can be used to create script-specific values.

importPreset(scriptIdentifier, preset)¶

This method should return variables and types of the concrete type which will be injected into the ScriptEngines scope.

unload(scriptIdentifier)¶

This will be called when the ScriptEngine will be unloaded (e.g. if the Script is deleted or updated). Script-specific information should be removed.
